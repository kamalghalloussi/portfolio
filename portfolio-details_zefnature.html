Information sur le projet

  • Type de site: Site E-commerce
  • Client: Zefnature
  • Date du projet: 01 Avril, 2022
  • URL du projet: www.zefnature.com

Dans ce projet, j'ai du présenter un magasin en ligne spécialisé dans la vente de professionnel de savon.j'ai utilisé le CMS WordPress pour faire la structure du site ainsi que l'extension Woocommerce pour gérer tous les paramétrages de la boutique. Le client en question a souhaité vendre une dizaine de produit tous repertoriés avec une fiche de description et variation de prix selon le nombre de commande. Un espace utilisateur a aussi été configuré sur mesure avec possibilité d'avoir accé au récapitulatif de commande, possibilité de télécharger les factures et le manuel d'installation pour chaque produit. Le site est 100% responsif adapte à toutes les formes d'écran (PC-mobile-tablette).
